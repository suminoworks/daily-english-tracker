# 英作文トラッカー 復習機能 設計書

## 概要

添削後にユーザー自身が「できた／できなかった」を評価し、
「できなかった」表現だけを復習対象として管理する仕組み。

---

## データ構造

### `savedItems` の各アイテム

```js
{
  id:          number,   // Date.now() で生成
  jp:          string,   // 日本語原文
  userEn:      string,   // ユーザーの英訳
  modelAnswer: string,   // 模範訳
  errors:      string,   // 改善ポイント（なければ空文字）
  phrases:     [{ en: string, ja: string }],  // 覚えるべき表現
  date:        string,   // 保存日 "YYYY-MM-DD"
  needsReview: boolean | undefined
  // true      = できなかった（要復習）
  // false     = できた（習得済み）
  // undefined = 旧データ（後方互換のため要復習扱い）
}
```

### 状態遷移

```
[新規問題]
    ↓ 添削
[自己評価]
    ├─ できなかった → needsReview: true  → 復習対象
    └─ できた       → needsReview: false → 復習から除外

[再チャレンジ後に再評価]
    ├─ できなかった → needsReview: true のまま（更新）
    └─ できた       → needsReview: false に更新 → 復習から除外
```

- 同じ `jp` のアイテムが既に存在する場合は**上書き更新**（重複させない）
- 保存は「できた」「できなかった」どちらを押しても行う（音読タブ用）

---

## タブ別 表示ルール

### 練習タブ

| テーマ | 問題文の取得元 |
|--------|---------------|
| セキュリティ Biz / ビジネス / 日常 / 旅行 / ニュース | Claude API（または内蔵サンプル） |
| **復習** | `savedItems` のうち `needsReview !== false` のものからランダム |

- 復習テーマ選択時、対象アイテムが0件 → 問題文エリアは空のまま（エラー表示なし）
- ボタンラベル: 復習テーマ時は「別の表現を選択する」、それ以外は「＋ 新しい問題文を生成」

### 復習タブ

- `needsReview !== false`（= `true` または `undefined`）のアイテムのみ表示
- 0件の場合: 「要復習の表現はありません。添削後に「できなかった」を押すと追加されます。」
- 各アイテムに「再チャレンジ ↗」ボタン → 練習タブへ遷移し問題文をセット

### 音読タブ

- `savedItems` の**全アイテム**を表示（`needsReview` 状態に関わらず）

---

## UI 変更点

### 添削結果カード（練習タブ）

**変更前**
```
[この表現を保存する]
[次の問題へ]
```

**変更後**
```
[できなかった]  [できた]
[次の問題へ]
```

- 「できなかった」: 赤系（red カラー）
- 「できた」: 緑系（teal カラー）
- どちらかを押したら両ボタンを `disabled` にする（押し直し不可）
- 「次の問題へ」は評価前でも押せる（未評価のまま次へ進める）

### 復習タブ

- バッジ表示:
  - `needsReview: true` → 「要復習」（amber）
  - `needsReview: undefined`（旧データ）→ 「未評価」（グレー）

---

## 決定事項

1. **「次の問題へ」を評価なしで押した場合** → 保存しない（未記録のまま）
2. **復習タブの並び順** → 保存日の新しい順（デフォルト降順）
3. **「できた」アイテムの確認** → 復習タブ内にトグルを設置

---

## 復習タブの詳細UI

タブ上部に「要復習 / 習得済み」トグルを設置。

```
[要復習 ●] [習得済み]   ← トグル
─────────────────────
（該当アイテム一覧）
```

| トグル状態 | 表示対象 |
|-----------|---------|
| 要復習（デフォルト） | `needsReview !== false` |
| 習得済み | `needsReview === false` |

- デフォルトは「要復習」
- 習得済みアイテムには「再チャレンジ ↗」ボタンを引き続き表示（要復習に戻せる）
